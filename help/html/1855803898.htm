FileService..::..GetFilesOnApp Method

Returns all the files related to the items in the application. This includes files both on the item itself and in comments on the item.

createdOn

The from and to date the file was created between. For valid operations see date filtering under the view area in Podio API documentation.
filetype
Type: String
The type of the file. Can be one of {"image", "application", "video", "text", "audio"}.
hostedBy
Type: String
Which provider actually hosts the files. Currently can be one of {"podio", "google", "boxnet", "dropbox", "evernote", "live", "sharefile", "sugarsync", "yousendit"}.
limit
Type: Int32
The maximum number of files to return Default value: 20
offset
Type: Int32
The offset to use when returning files to be used for pagination. Default value: 0
