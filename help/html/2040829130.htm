"incoming" to get incoming notifications, "outgoing" to get outgoing notifications. Default value: incoming

The maximum number of notifications to return, maxium is "100". Default value: 20
offSet

The offet into the returned notifications. Default value: 0
starred
Type: Nullable<(Of <(<'Boolean>)>)>

viewed

Type: Nullable<(Of <(<'DateTime>)>)>
